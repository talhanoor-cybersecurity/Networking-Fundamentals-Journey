## 🔹 What I Learned

### Router
A router connects different networks and forwards data between them.

### Switch
A switch connects devices within the same network and sends data to the correct destination.

### Hub
A hub connects multiple devices and broadcasts data to all connected devices.

### Modem
A modem connects a network to the Internet through an Internet Service Provider (ISP).

### Access Point
An access point provides wireless connectivity and allows devices to connect to a network via Wi-Fi.
